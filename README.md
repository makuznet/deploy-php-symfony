# Postgresql migration PHP

> This repo creates Postgresql migration with PHP.   

Composer - Dependency Manager for PHP.


## Usage 
### Run Composer
```bash
composer
```
## Installation
### PHP on MacOS
```bash
brew install php
```
### Composer on MacOS
```bash
brew install composer
```
- [Download Composer](https://getcomposer.org/download)
